# Bus Ticket Management System

A simple console-based Python application to manage bus ticket records using MySQL database. It supports full CRUD operations for easy booking and management of bus tickets

---

## Features

- Book new bus tickets
- View all existing ticket records
- Update ticket details
- Delete ticket records
- Automatic connection handling to MySQL database
- Input validation and error handling
- Data integrity using commit and rollback
- Shows booking date and time automatically

---

## Technologies Used

- **Python 3**
- **MySQL / phpMyAdmin**
- **Library:** `mysql-connector-python`

---

## How to Run

### 1. Install required library

Open your terminal or command prompt and run:

```bash
pip install mysql-connector-python
```
